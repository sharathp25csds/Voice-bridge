Breaking Communication Barriers with AI

VoiceBridge is an AI-powered communication platform designed to help people with speech and hearing disabilities communicate more effectively and confidently in real time.

The project combines Speech Recognition, Text-to-Speech, Live Call Captioning, and AI Assistance into one accessible and modern web experience.

✨ Features
🎤 Speech-to-Text (STT)

Convert live voice into readable text instantly.

✅ Real-time voice recognition
✅ Fast and responsive
✅ Easy communication support

🔊 Text-to-Speech (TTS)

Convert typed text into natural voice output.

✅ Clear voice playback
✅ Helps non-verbal communication
✅ Simple and user-friendly

📞 Live Call Captioning

Generate real-time captions during conversations and calls.

✅ Live subtitles for calls
✅ Better accessibility
✅ Improved hearing assistance

🤖 AI Assistant

Smart AI support for faster and easier communication.

✅ AI-generated responses
✅ Communication suggestions
✅ Accessibility-focused assistance
